Privacy Policy for Codeforces Daily Momentum

Effective Date: June 2026

1. Information Collection and Use
"Codeforces Daily Momentum" (the "Extension") operates entirely locally within your browser. The Extension does NOT collect, store, track, or transmit any personally identifiable information (PII).

2. Codeforces API Data
To generate the visual dashboard, the Extension fetches public submission data directly from the official Codeforces API (https://codeforces.com/api/). This data is strictly used to render the graphs and calculate the "Momentum Power Score" within the user interface.

3. Local Storage
The Extension may temporarily cache public Codeforces API responses locally in your browser (using chrome.storage.local) solely to improve loading speeds and reduce API calls. This data never leaves your device and is never sent to any external server or third party.

4. Third-Party Sharing
We do not sell, trade, or otherwise transfer your data to outside parties. There are no tracking scripts, analytics, or ads embedded in this Extension.

5. Contact
If you have any questions regarding this privacy policy, you may contact the developer via the support email listed on the Chrome Web Store.
